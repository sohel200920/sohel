Sohel Imam Shaikh — Portfolio (static)
------------------------------------
This is a static, responsive one-page portfolio built for Sohel Imam Shaikh.
Replace the placeholder images in the /assets folder with real screenshots and profile photo.

How to use:
1) Unzip the folder.
2) Open index.html in a browser to preview.
3) Deploy to GitHub Pages / Netlify / Vercel by pushing the folder to your repository.
4) Update contact email, GitHub and LinkedIn links in index.html.

Files:
- index.html
- styles.css
- script.js
- assets/ (placeholders)
- resume.pdf (sample — replace with your own)
